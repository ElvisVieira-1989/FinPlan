FUNCIONALIDADES

• Planejamento Simplificado: Insira seu salário mensal e quanto deseja investir para obter um cálculo instantâneo.<br>
• Composição de Gastos: Sugestão baseada na regra 50/30/20 (adaptada), dividindo seu orçamento em: <br>
   - Gastos Essenciais: O que você precisa para viver.<br>
   - Estilo de Vida: Seus gastos com lazer e desejos pessoais.<br>
   - Investimentos: Foco na construção de patrimônio.<br>


• Perfis de Investimento: Sugestões detalhadas de alocação de ativos em três níveis:

-🟢 Conservador: Foco em segurança e liquidez (Tesouro Selic, CDB).<br>
-🟡 Moderado: Equilíbrio entre renda fixa e variável.<br>
-🔴 Agressivo: Busca por retornos maiores com exposição à volatilidade (Ações, Cripto).<br>

• Inteligência de Localização: O app detecta automaticamente se você está no Brasil para validar a configuração da moeda em Reais (BRL).<br>
• Design Moderno: Interface construída com Material Design 3, suporte a Tema Escuro e visual Edge-to-Edge.

TECNOLOGIS UTILIZADAS

• Linguagem: Kotlin<br>
• UI: Jetpack Compose (Declarativa)<br>
• Arquitetura: MVVM (Model-View-ViewModel)<br>
• Estilo: Material 3 com esquema de cores vibrante.<br>
• Localização: Google Play Services Location & Geocoder.<br>
• Permissões: Accompanist Permissions para gestão de acesso à localização.<br>
• Assincronismo: Kotlin Coroutines & Flow.<br>

COMO EXECUTAR O PROJETO

1. Clone este repositório:
Shell Script: git clone https://github.com/ElvisVieira-1989/finplan.git
2. Abra o projeto no Android Studio (versão Ladybug ou superior recomendada).
3. Sincronize o Gradle.
4. Execute o app em um emulador ou dispositivo físico com Android 8.0 (API 26) ou superior.

PERMISSÕES

O aplicativo solicita as seguintes permissões:<br>
• ACCESS_COARSE_LOCATION: Para identificar o país de origem.<br>
• INTERNET: Para serviços de Geocoder e verificação de rede.<br>
